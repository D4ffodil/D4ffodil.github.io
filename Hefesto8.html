Classes e IDs

Podemos dar uma classe elementos, quando queremos classifica-los com um estilo específico. Podemos dar a mesma classe para vários elementos, eles compartilhrão do mesmo estilo.

Podemos dar um IDa elementos, qauando queremos estiliza-lo individualmente. Não podemos dar o mesmo ID para elementos diferentes.
